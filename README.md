An interactive and responsive Tic Tac Toe web application built using HTML, CSS, and JavaScript. The game features both Single Player (vs Computer) and Multiplayer modes, along with a modern glassmorphism user interface, score tracking, winner popup notifications, and smooth animations for an engaging gaming experience.
